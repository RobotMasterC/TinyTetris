# TinyTetris

A small console that plays Tetris that is powered by an Attiny85

## BOM
- 1x Attiny85-20u
- 1x 128x32 I2C screen
- 3x Buttons
- 1x Power Switch
- 1x 3024 Button Battery Holder (For 2032 Battery)
